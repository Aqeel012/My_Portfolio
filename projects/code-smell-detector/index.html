Code Smell Detector

Code Smell Detector is a developer tool designed to analyze source code and identify maintainability issues through static code analysis. The tool detects common code smells such as large classes, excessively long methods, long parameter lists, and useless exception handling blocks. By analyzing the structure of source code files, the system highlights problematic patterns and generates reports to help developers refactor and improve code quality. The project demonstrates practical implementation of static analysis techniques for improving software maintainability.

GitHub

Key Features

  • Detection of Large Class code smell
  • Detection of Long Method code smell
  • Detection of Long Parameter List in functions
  • Identification of useless or unnecessary exception handling
  • Source code scanning and analysis
  • Report generation listing detected code smells

Challenges & Solutions

Implementing accurate detection logic for multiple code smells required analyzing code structure using AST parsing while minimizing false positives during detection.
